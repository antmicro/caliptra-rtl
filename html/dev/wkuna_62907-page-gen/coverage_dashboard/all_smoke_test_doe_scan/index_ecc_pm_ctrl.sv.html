      19              : // ECC point multiplication controller for the Secp384.
      20              : //
      21              : // The architecture includes ecc_pm_sequencer including the required 
      22              : // sequence to perform:
      23              : //   - point multiplication (pm) based on Montgomery ladder and Fermat's little 
      24              : //    theorem (FLT) inversion
      25              : //   - Montgomery ladder initilization with base point G or public key PK in
      26              : //     keygen, signing, or verifying operations.
      27              : //   - Montgomery ladder steps: including the iterative point addition (PA)
      28              : //     and point doubling (PD) operations in Montgomery domain using projective 
      29              : //     coordinates. The number of iteration is equal to REG_SIZE of scalar in 
      30              : //     unprotected point multiplication, and REG_SIZE+RND_SIZE of scalar in the 
      31              : //     SCA protected operation.
      32              : //   - Modular Inversion (mod p, mod q) is based on Fermat's little theorem (FLT) 
      33              : //     inversion.
      34              : //   - Conversion from projective coordinates in Montgomery domain (X,Y,Z) to 
      35              : //     affine coordinates in the normal domain (x,y).
      36              : //   - Modular arithmetic operations such as (h + r*privKey) in signing
      37              : //     or (h*s_inv) and (r*s_inv) in verifying
