      17              : // ecc_dsa_sequencer.sv
      18              : // --------
      19              : // This module contains the sequence of ECC operation. it includes 4 sequences
      20              : // of operation:
      21              : // 1) reset sequence:  After reset, the core will initilze the data memory based
      22              : //                     on the first lines of sequencer. Then, it will stay on NOP
      23              : //                     line until getting the command.
      24              : // 2) keygen sequence: After getting the keygen command, it will generate and 
      25              : //                     randomize privkey and feed it into the pm core and return
      26              : //                     key pairs of (privkey, pubkey)
      27              : // 3) sign sequence:   After getting the sign command, it will generate and 
      28              : //                     randomize nonce and feed it into the pm core and return
      29              : //                     the signature (R, S).                     
      30              : // 4) verify sequence: After getting the verify command, it will feed the signature
      31              : //                     into the pm core and return the computed verify_R.
      32              : //
      33              : //
