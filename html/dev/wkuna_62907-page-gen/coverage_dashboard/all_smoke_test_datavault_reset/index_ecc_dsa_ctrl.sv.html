      17              : // ecc_dsa_ctrl.sv
      18              : // --------
      19              : // Elliptic Curve Cryptography (ECC) digital signature algorithm (DSA) 
      20              : // controller to support deterministic ECDSA based on RFC 6979.
      21              : // The dsa architecture includes several countermeasuress to be protected
      22              : // against a subset of side-channel analysis (SCA) attacks. 
      23              : //
      24              : // The architecture includes:
      25              : // 1) ecc_dsa_sequencer: including the required sequence to perform 
      26              : //    keygen, signing, and verifying.
      27              : // 2) ecc_arith_unit: the arithmetic unit to perform point multiplication
      28              : //    and other required operations
      29              : // 3) hmac_drbg: the hmac384 drbg component to support RFC 6979
      30              : // 4) ecc_scalar_blinding: the SCA countermeasure to randomized scalar
      31              : //    to avoid information leakage.
      32              : //
      33              : // The embedded countermeasures are:
      34              : // - point randomization: 
      35              : //      Randomized Projective Coordinates countermeasure based on section 5.3 
      36              : //      "Resistance against Differential Power Analysis for Elliptic Curve 
      37              : //      Cryptosystems" in CHES 1999 by Coron, J.  
      38              : //      This countermesure uses a random value "lambda" in point multiplication
      39              : //      (Q = k * P) in keygen/signing to randomiza base point P = (X, Y, 1)
      40              : //      as follows:
      41              : //      P = (X * Lambda, Y * lambda, Lambda)
      42              : // - scalar blinding: 
      43              : //      Scalar blinding countermeasure based on section 5.1 
      44              : //      "Resistance against Differential Power Analysis for Elliptic Curve 
      45              : //      Cryptosystems" in CHES 1999 by Coron, J.  
      46              : //      This countermeasure uses a random value "r" in the point multiplication
      47              : //      (Q = k * P) in keygen/signing to randomiza scalar k as follows:
      48              : //      k = k + r*group_order
      49              : // - masking signature: 
      50              : //      Masking sign countermeasure uses a random value "d" in the signing 
      51              : //      operation to generate signature proof s = (privkey * r + h)*k_inv 
      52              : //      as follows:
      53              : //      s = [((privkey - d) * r + (h - d)) * k_inv] + [(d * r + d) * k_inv
