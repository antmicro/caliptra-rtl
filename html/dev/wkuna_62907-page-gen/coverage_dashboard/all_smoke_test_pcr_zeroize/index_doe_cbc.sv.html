      19              : // The module supports DOE_CBC and it is the modified version of doe.v file. 
      20              : // In the hierarcy, one level up module is the doe_ctrl_32.sv or doe_ctrl_64.sv

      22              : // key, and IV.

      24              : // The "IV XOR Plaintext" operation is just added into the DOE first Key XOR path.
      25              : // The combinational logical path is ((IV XOR Plaintext) XOR Key). The lower
      26              : // hierarcy, doe core, has not been changed. Instead, doe_core_cbc was created.

      29              : // DOE CBC mode is tested with doe_cbc_tb.sv
